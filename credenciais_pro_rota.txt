📋 FUNCIONALIDADES DO SISTEMA
================================================================================

✅ RASTREAMENTO GPS EM TEMPO REAL
   - Localização do usuário no mapa (marcador azul)
   - Botão "Minha Localização" para centralizar o mapa
   - Zoom + e - para ampliar/reduzir o mapa
   - Posição dos ônibus em movimento (marcadores verdes)
   - Velocidade em tempo real (km/h)
   - Precisão do sinal GPS (metros)

✅ CHECKLIST DO MOTORISTA
   - Verificação de sonolência (Não/Leve/Moderada/Intensa)
   - Aptidão para dirigir (Sim/Não)
   - Condições adicionais (cansado, medicamento, dores)
   - Bloqueio automático se:
     * Motorista marcar "não apto"
     * Sonolência = "intensa"

✅ DASHBOARD OPERACIONAL
   - Cards com estatísticas em tempo real
   - Mapa com todos os veículos ativos
   - Lista de ônibus em operação
   - Status atualizado automaticamente

✅ PORTAL DO PASSAGEIRO
   - Visualização pública dos ônibus
   - Busca por número do ônibus
   - Previsão estimada de chegada
   - Localização dos veículos no mapa

✅ CENTRAL DE ALERTAS
   - Checklist recusado
   - Motorista inapto
   - GPS sem sinal
   - Troca de motorista
   - Velocidade excessiva

⚙️ CONFIGURAÇÕES NECESSÁRIAS NO CELULAR:

1. PERMISSÃO DE LOCALIZAÇÃO:
   - Android: Configurações → Apps → Navegador → Permissões → Localização → Permitir
   - iOS: Configurações → Privacidade → Localização → Navegador → Permitir

2. GPS ATIVADO:
   - Ativar o GPS nas configurações do celular
   - Modo de alta precisão recomendado


🎯 COMO UTILIZAR O SISTEMA
================================================================================

🚀 PARA MOTORISTAS:

1. Acesse o sistema no navegador do celular
2. Faça login com MATRÍCULA: MEDE / SENHA: 123456
3. Clique no menu "Checklist"
4. Preencha todas as perguntas do formulário
5. Clique em "Iniciar Rota e Ativar GPS"
6. PERMITA o acesso à localização quando solicitado
7. Pronto! O sistema começará a rastrear sua rota

👁️ PARA SUPERVISORES/ADMIN:

1. Acesse pelo computador ou tablet
2. Faça login com credenciais de administrador
3. Visualize o Dashboard com todos os ônibus
4. Use o botão "Minha Localização" para se encontrar


🔧 SOLUÇÃO DE PROBLEMAS
================================================================================

❌ PROBLEMA: GPS não funciona
✅ SOLUÇÃO: 
   - Verifique se o navegador tem permissão de localização
   - Recarregue a página e clique em "Minha Localização"

❌ PROBLEMA: Login não funciona
✅ SOLUÇÃO:
   - Verifique se a matrícula e senha estão corretas
   - Use exatamente: MEDE / 123456 para teste
   - Verifique se as credenciais estão no arquivo TXT
